Add to HomeIndexr — Options

Point the extension at your HomeIndexr instance. Adding a property sends the listing's address to this server, which matches it against Realtor.com and captures a first snapshot.

Saved

Default is http://127.0.0.1:5173 (what ./run.sh serves). The extension can only reach localhost / 127.0.0.1 hosts out of the box — a custom host needs the matching entry added to host_permissions in manifest.json.
